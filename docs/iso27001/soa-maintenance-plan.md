# ISO 27001 Statement of Applicability (SoA) Maintenance Plan

## Review cadence
- Quarterly review of control status and justification.
- Update on major changes (new systems, vendors, or regulations).

## Update workflow
1. Run quarterly gap analysis.
2. Update control status and evidence links.
3. Review exceptions and residual risk.
4. Approve via management review.

## Evidence
- Updated SoA document
- Change log with rationale
- Management review minutes
